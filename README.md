# Cleveroad-Web-Test
Web application that displays in real time the current location of the ISS, as well as the size and composition of its crew.

## Project setup

```bash
npm install
```

## Usage

```bash
npm run start
```

## Compiles and hot-reloads for development

```bash
npm run dev
```

## Compiles and minifies for production

```bash
npm run build
```
